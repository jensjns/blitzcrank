== Description ==

Jump around wp-admin in a blitz! Show the help-section by pressing <code>b</code> then <code>?</code>

If you are using WordPress' admin-section in english keyboard-shortcuts to the top-level pages will be:  
`d a` Dashboard
`p o` Posts
`m e` Media
`p a` Pages
<code>c o</code> Comments
And so on.. It uses the first two letters of the label to create the keyboard shortcut.

== Installation ==

1. Upload the `blitzcrank` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Check out the help-section by pressing `b ?` on your keyboard
